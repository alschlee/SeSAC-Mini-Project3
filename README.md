# SeSAC Cloud Camp Mini Project 3

## Project Overview

AWS를 활용하여 서비스 인프라를 구성하고, 응용 프로그램을 배포하는 프로젝트입니다. <br/>


## Project Duration

- **기간**: 10월 14일 (목) ~ 10월 21일 (월)

## Requirements

- 프론트엔드, 백엔드, 데이터베이스로 구성
- 데이터베이스는 RDS를 사용
- 수업 시간에 사용했던 리소스 범주 내에서 구성
